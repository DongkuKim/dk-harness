# Product Sense

This repo is optimized for agents and developers who want a small, opinionated scaffold catalog for Next.js, Python, and Rust projects.

The default experience should communicate:

- the supported starters are intentionally small and explicit
- template selection is based on visible stack, runtime, and lane metadata
- scaffolding copies the committed template exactly
- release and CI checks are part of the product promise, not just internal maintenance

Detailed specs live in `product-specs/`.
